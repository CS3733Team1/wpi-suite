Summary:

The filter class represents a calendar filter. Filters are objects that have a list of allowed categories. Filters also have names. Filters have their own list panel on the Janeway Graphical User Interface which lists all of the filters.

How To:

Add a Filter

In order to add a filter, first locate and select the filter tab on the rightmost panel of the GUI.

Once selected, the panel will show two buttons and a collection of radio buttons below. Click the button that says Add Filter.

The panel will change to show the components used to create a filter. Enter a name in the appropriate textbox and select the categories you would like to see once the filter is applied. Multiple categories can be selected by holding ctrl/command and clicking on the desired categories. Once the selections are made, click the Add Category button. The selected categories will move to the lower box. Verify that these are the categories that should be shown once the filter is applied, and click the OK button.

Delete a Filter

To delete a filter, first locate the filter on the Filter List Panel. Click it once so that the filter is highlighted.

Once the filter is highlighted, locate the Delete Filter button in the Filter List Panel. Press this button to delete the commitment.

Multiple filters can be deleted at a time. To delete multiple filters, first select all of the filters to be deleted. To select multiple filters, hold the ctrl/command key and single click on the desired filters.

Once the filters are selected, locate the Delete Filter button. Press this button to delete the filter.
