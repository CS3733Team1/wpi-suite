Summary:

The commitment class represents a calendar commitment. Commitments are objects that have a due date and time, but not a start date or time. Commitments also have names, categories, and/or a description. Commitments have their own list panel on the Janeway Graphical User Interface which lists all of the commitments. A user can double click on a commitment to exchange the list view for a detail view. In the detail view, the user is given the option to either edit the commitment or return to the list view. Should the user choose to edit a commitment, a new tab will open and auto-populate with the commitments information. The user can then change the details of the commitment and save it.

How To:

Add a Commitment

In order to add a commitment, first locate and click on the Add Commitment Button.

A new tab will appear. Enter the appropriate information in each of the fields and press the Add Commitment button.

The new commitment will appear on the calendar and in the Commitment List Panel.

Edit a Commitment

To edit a commitment, first locate the commitment on the Commitment List Panel. Double click on the commitment.

This brings up a detail view of the commitment. Locate the edit commitment button near the top of the panel.

Once the edit commitment button has been pressed, a new tab will appear. The fields will auto-populate with details about the commitment. Edit the desired fields and press the Update Commitment button. This will close the tab and bring you back to the standard view.

Delete a Commitment

To delete a commitment, first locate the commitment on the Commitment List Panel. Click it once so that the commitment is highlighted.

Once the Commitment is highlighted, locate the Delete Commitment button. Press this button to delete the commitment.

Multiple commitments can be deleted at a time. To delete multiple commitments, first select all of the commitments to be deleted. To select multiple commitments, hold the ctrl/command key and single click on the desired commitments.

Once the Commitments are highlighted, locate the Delete Commitment button. Press this button to delete the commitments.
