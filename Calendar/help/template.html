Adding Text

Surround help text with paragraph tags ("p" in angle brackets). Each section should have a heading describing the section's subject. Remember that linebreaks don't show up unless you use a "br" tag.

Adding Images to Text

Images can be added with an image tag. When adding an image, make sure you copy the image file into the help folder (this document's parent directory) and refer to an image as a relative path.
Be sure that you precisely follow the format of image tags given in the example below. Otherwise, the parser will break.

Adding Headings

A heading is marked with the "hx" tag, where x is a positive integer which describes the level of nesting of the heading. That is, headings provide a way to organize a document into a tree; h1 headings are root nodes, h2 headings should be children of h1 sections, h3 headings children of h2, and so on.
