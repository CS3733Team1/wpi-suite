Summary

Events are schedulable happenings that have a duration. Events have names, start dates, end dates, categories, participants and descriptions. Events are shown on the various calendar views.

Adding an Event

In order to add an event, first locate and click on the Add Event Button.

A new tab will appear. Enter the appropriate information in each of the fields and press the Add Event button.

The new event will appear on the calendar.

Editing an Event

To edit an event, first locate the event on the calendar. Double click on the event.

This brings up a new tab. The fields will populate with existing details about the event. Edit the desired fields and press the Update Event button. This will close the tab and bring you back to the standard view.

Deleting an Event

To delete an event, first locate the event on the calendar. Click it once so that the event is highlighted.

Once the event is highlighted, locate the delete button. Press this button to delete the event.

Multiple events can be deleted at a time. To delete multiple events, first select all of the events to be deleted. To select multiple events, hold the Ctrl/command key and single click on the desired events.

Once the events are highlighted, locate the delete button. Press this button to delete the events.
