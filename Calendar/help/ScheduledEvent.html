Summary

Scheduled Events are schedulable happenings that have a duration. Scheduled Events have event names, event start time and end time, and various days of the week. Scheduled Events are shown on the various calendar views.

Schedule an Event

In order to Schedule event, first locate and click on the Add Scheduled Event Button on the upper left corner of the of the calendar next to add event button .

A new tab will appear to poppulate. Enter the appropriate information in new event fields and the start time and end time field. To select a specific day, simple move your cursor and click on specific days you wish to select and they will be highlighted. Now, press add event button at the lower left corner of the window which will redirect you to a new window to show up your selected days.

Updating an Event

To update an event from the scheduled event window, first, locate the update event button on the lower left corner of the window and press it.

The update event button will close the tab and bring you back to a new add event tab to update the event. To update an event, please refer to the instructions on how to add events.
