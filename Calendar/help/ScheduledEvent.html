Summary

Scheduled events help you find the best time for a group to meet. It is compact, so that events can be created and accessed quickly. Scheduled events have event names, event start times and end times, and days of the week to occur on. In Scheduled Events, the event owner selects a date and time range, then the users block out the times they can't meet. The best times with the most available people are visually highlighted by color.

Schedule An Event

In order to create a scheduled event, first locate and click on the New Scheduled Event Button on the upper left corner of the calendar next to the New Event button.

A new tab will appear to populate. Enter the appropriate information in new event fields and the start time and end time field. To select a specific day, simple move your cursor and click on specific days you wish to select and they will be highlighted.

Now, press add event button at the lower left corner of the window which will redirect you to a new window to show up your selected days.

Updating An Event

To update an event from the scheduled event window, first, locate the update event button on the lower left corner of the window and press it. The update event button will close the tab and bring you back to a new add event tab to update the event. To update an event, please refer to the instructions about add events.
